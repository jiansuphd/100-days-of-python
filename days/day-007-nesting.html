The Nesting Doll

Nesting is where we put an if statement within another if statement using the power of indenting.

Think of it like a decision tree: "If I am hungry (Yes)... then: Do I want pizza (Yes/No)?" You only ask the second question if the first one is true! üê¶ü•ö

üëâ The Indentation Rule

The second if statement (the baby bird) must be indented inside the first one.

Any print statement belonging to that second if must be indented one more time.

Pay close attention to the vertical lines in your code editor. They show you which statement belongs to which block!

üíª Syntax: TV Show Logic

Delete any old code in main.py and try this example. Do you notice the second if statement about fave_character?

tv_show = input("What is your favorite tv show? ")
if tv_show == "peppa pig":
  print("Ugh, why?")
  fave_character = input("Who is your favorite character? ")
  if fave_character == "daddy pig":
    print("Right answer")
  else:
    print("Nah, Daddy Pig's the greatest")
elif tv_show == "paw patrol":
  print("Aww, sad times")
else:
  print("Yeah, that's cool and all...")

Code Analysis:

  • First Layer: We check the TV show first.
  • Second Layer: ONLY if the show is "peppa pig", do we ask about the character.
  • The Indent: Notice how the input for fave_character is indented? It lives inside the first if.
