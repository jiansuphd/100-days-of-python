ðŸ“¦ What is a Variable?

Think of input() like someone handing you an object. If you don't have anywhere to put it, you have to drop it immediately.

A variable is like a box with a label on it. We can take the input and store it in the box to use later.

Naming Rules: You can give a variable almost any name, but:
  • No spaces allowed!
  • Use underscores_between_words
  • Use camelCaseToMakeItReadable

ðŸ’» Syntax: Capturing Input

Delete your old code in main.py and try this:

myName = input("What's your name?: ")
myAge = input("How old are you? :")
print("Gee, that's REALLY OLD")
replit = input("Do you like Replit?")
print("OF COURSE YOU DO!")

Breakdown:

  • myName is the Variable (the box).
  • = is the Assignment operator (putting the value in the box).
  • input(...) is the Function (getting the value from the user).
