ðŸ¦¸ The Print Super-Power

You just made a full sentence, right? But how did it work?

It turns out print has a super-power. We can give it as many different things to print as we want.

Rule: Put a comma , between every different thing inside the ().

ðŸ’» Syntax: The Comma Trick

Delete any old code in main.py and try this example:

myName = input("What's your name? ")
myLunch = input("What are you having for lunch? ")
print(myName, "is going to be chowing down on", myLunch, "very soon!")

What happens?

  • The computer prints the value of myName.
  • It automatically adds a space where the comma is.
  • It prints the text string.
  • It adds another space, then myLunch.
