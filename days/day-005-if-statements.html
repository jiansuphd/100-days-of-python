‚ùì Is this equal?

When we want to check if two things are exactly the same, we use the double equals sign ==.

myName == "David"
Is the variable myName the same as the string "David"?

üíª Syntax: The Indent Matters

Here is how we write it. Pay close attention to the colon : and the indentation.

myName = input("What's your name?: ")

if myName == "David":
  print("Welcome Dude!")
  print("You're the baldest dude I've seen")
else:
  print("Who on earth are you?!")

How it works:

  • The Condition: Checks if the name is "David".
  • The Indent: To create a print statement related to the if, you must go to the next line and indent once.
  • The Else: If the condition is not met, the computer jumps to the else block.
  • Alignment: else must be unindented, lining up perfectly with if.
